# WasteWatch – Prompts Log

Registo dos principais prompts utilizados com ferramentas de IA durante o desenvolvimento do projeto.

## Prompt 1

**Goal:**
Generate all documentation markdown files for the WasteWatch project based on the project proposal.

**Prompt used:**
"preciso que me ajudes a definir o conteúdo dos ficheiros .md para a poder desenvolver a minha aplicação wastewatch" (com upload do Project Proposal PDF e screenshot da estrutura de ficheiros requerida)

**Result:**
Geração de todos os ficheiros de documentação: `01_overview.md`, `02_features.md`, `03_screens.md`, `04_data_model.md`, `05_navigation.md`, `06_architecture.md`, `07_api_usage.md`, `08_implementation_plan.md`, `agents.md`, `README.md`.

Adapt and refine all markdown files to follow the exact structure and conventions specified in the practical assignment tutorial.
